# stm32-rc-with-mecanumwheels
Mecanumwheel RC Car (with STM32F103 & NRF24L01P)

STM32F103C8T6 - 메카넘휠 RC카입니다.  
NRF24L01P모듈이 수신한 데이터로 DC모터 4개를 제어하여 전/후진, 회전, 측면 이동을 구현

----------------------------------------------------------------------------------

기능 요약
- NRF24L01P 무선 모듈을 통한 데이터 수신
- L298N 모터 드라이버로 DC기어박스 모터 제어
- 메카넘휠을 이용한 다방향 이동 (전/후진, 좌/우 이동, 회전)
- 디버깅 및 테스트를 위한 시리얼 출력 지원
- 모터 속도 제어를 위한 PWM 신호 사용
- 간단한 체크섬을 통한 데이터 무결성 확인

----------------------------------------------------------------------------------

사용 부품 / 모듈 목록

STM32F103C8T6 (Blue Pill) x1 - 메인 MCU
NRF24L01P 무선 모듈 x1 - 데이터 수신
L298N 모터 드라이버 x2 - DC모터 제어 
DC 기어박스 모터 x4 - 메카넘휠 구동
메카넘휠 세트 x1 - 이동 구현
세라믹 캐패시터 0.1uf x4 - 모터 노이즈 필터링
전해 캐패시터 10uf x1 - nrf24 안정화
점퍼 케이블, 브레드보드 등 - 배선 및 연결
2s LiPo 배터리 팩 x1 - 전원 공급

----------------------------------------------------------------------------------
